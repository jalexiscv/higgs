Ayudante de seguridad

El archivo Security Helper contiene funciones relacionadas con la seguridad.

  • Cargando este ayudante

  • Funciones disponibles

Cargando este ayudante

Este ayudante se carga usando el siguiente código:

<?php

helper('security');

Funciones disponibles

Están disponibles las siguientes funciones:

sanitize_filename($filename)
Parámetros:

  • archivo (string $nombre de) – nombre de archivo

Devuelve:

nombre de archivo desinfectado

Tipo del valor devuelto:

cadena

Proporciona protección contra el cruce de directorios.

Esta función es un alias para \Higgs\Security::sanitizeFilename(). Para obtener más información, consulte la Biblioteca de seguridad documentación.

strip_image_tags($str)
Parámetros:

  • $cadena (string) – cadena de entrada

Devuelve:

la cadena de entrada sin etiquetas de imagen

Tipo del valor devuelto:

cadena

Esta es una función de seguridad que eliminará las etiquetas de imágenes de una cadena. Deja la URL de la imagen como texto sin formato.

Ejemplo:

<?php

$string = strip_image_tags($string);
encode_php_tags($str)
Parámetros:

  • $cadena (string) – cadena de entrada

Devuelve:

cadena formateada de forma segura

Tipo del valor devuelto:

cadena

Esta es una función de seguridad que convierte etiquetas PHP en entidades.

Ejemplo:

<?php

$string = encode_php_tags($string);
